# STAR Method

## What is STAR?

A structured approach to answering behavioral questions:

| Letter | Meaning | What to Include |
|--------|---------|-----------------|
| **S** | Situation | Set the context and background |
| **T** | Task | Describe your responsibility |
| **A** | Action | Explain what YOU did (use "I", not "we") |
| **R** | Result | Share the outcome with metrics if possible |

## Tips

1. Keep answers 1-2 minutes long
2. Focus on YOUR contributions
3. Quantify results when possible
4. Prepare 5-6 stories that cover multiple questions
5. Practice out loud!

## Template

```
SITUATION: [Brief context - 1-2 sentences]
TASK: [Your specific responsibility - 1 sentence]
ACTION: [Steps you took - 3-4 sentences]
RESULT: [Outcome with numbers/impact - 1-2 sentences]
```
