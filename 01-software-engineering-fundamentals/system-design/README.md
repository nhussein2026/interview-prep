## Interview Framework

1. **Requirements** (5 min) - Clarify functional and non-functional
2. **Estimation** (5 min) - Scale, storage, traffic
3. **High-Level Design** (10 min) - Main components
4. **Deep Dive** (20 min) - Critical components
5. **Wrap Up** (5 min) - Bottlenecks, improvements
