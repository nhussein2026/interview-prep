## Core Concepts

### Four Pillars
1. **Encapsulation** - Bundle data and methods, hide internals
2. **Abstraction** - Hide complexity, expose interface
3. **Inheritance** - Create classes from existing ones
4. **Polymorphism** - Same interface, different behavior

### SOLID Principles
| Principle | Focus |
|-----------|-------|
| **S**ingle Responsibility | One reason to change |
| **O**pen/Closed | Extend, don't modify |
| **L**iskov Substitution | Subtypes substitutable |
| **I**nterface Segregation | Small interfaces |
| **D**ependency Inversion | Depend on abstractions |
